153def lookup_symbol_by_ticker(ticker)-> list: 

154 """Returns search dictionary for ticker/ 

155 sometimes the name of the ticker in the xlsx  

156 sheet is off slightly and we need to find the "real ticker" 

157 

158 Uses standard api to grab tickers 

159 TODO: make another function that uses the new graphql api instead 

160 in case this one gets depreciated 

161 Example searchpoint is https://app.quotemedia.com/lookup?callback=html&q=zmd&limit=5&webmasterId=101020 

162 

163 See https://app-money.tmx.com/graphql and https://money.tmx.com/en/ 

164 Ex: {"operationName":"findCompaniesByKeywords","variables":{"keywords":"zmd"},"query":"query findCompaniesByKeywords($keywords: String) {\n findCompaniesByKeywords(keywords: $keywords) {\n symbol\n name\n exchange\n __typename\n }\n}\n"} 

165 

166 Input: Ticker as string 

167 

168 Output: Dictionary 

169 """ 

170 if ticker is None: 

171 return [] 

172 callback = 'tmxtickers' 

173 # Theses values can be hardcoded 

174 params = { 

175 'callback': callback, 

176 'limit': 5, 

177 'webmasterId': 101020, 

178 'q': ticker 

179 } 

180 quote_lookup_url = 'https://app.quotemedia.com/lookup' 

181 r = requests.get(quote_lookup_url, params=params) 

182 quote_data = r.text 

183 

184 if quote_data is None: 

185 print('Data missing, no response available') 

186 s_idx = quote_data.find('(') 

187 e_idx = quote_data.rfind(')') 

188 

189 if quote_data == f"{callback}();": 

190 return [] 

191 # All symbols returned all valid 

192 try: 

193 data = loads(quote_data[s_idx + 1:e_idx]) 

194 return data 

195 except Exception as e: 

196 print(ticker) 

197 print('fail to run') 

198 print(e)
